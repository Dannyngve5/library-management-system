# Library Management System

A backend application for managing a library, developed in Python with a strong focus on clean software architecture and maintainability.

## Objectives

- Manage books and users
- Handle book loans and returns
- Track loan history
- Support multiple copies of each book
- Persist data using SQLite

## Architecture

This project follows a layered architecture inspired by Clean Architecture principles.

### Technologies

- Python
- SQLite

### Design Patterns

- Repository Pattern
- Unit of Work
- Factory Pattern
- Dependency Injection
- DTO (Data Transfer Objects)

### Project Structure

- Presentation
- Services
- Repositories
- Domain Models
- DTOs
- Unit of Work
- Database
- Migrations
